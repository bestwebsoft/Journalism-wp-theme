== Installation ==

Manual installation:

1. Upload the `Journalism` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress

Installation using "Add New Theme":

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'newlife'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button
5. Activate the theme through the 'Themes' menu in WordPress

== Theme Notes ==

= Slider =

For the slider you need to complete the following 3 steps:
1. Go to Dashboard > Posts, open a post, which you want to see in the slider, in the edit mode.
2. Set a checkmark in the checkbox (on the right side of the screen) and select the featured image for the post.
3. Update your post.
We recommend you to use the proportions 900x218px (width/height) for images which you want to add to the slider.

= Featured Images =

For featured images of posts or pages we recommend using the proportions 620x195px (width/height).

= Widgets =

The Theme includes standard widgets.
By default in the sidebar the following widgets will be displayed: search form, recent posts, recent comments, archive and categories.
You can change the set of widgets up to your choice.

== Frequently Asked Questions ==

= How to use the custom header image? =

You can use the custom header image the following way: from your Admin UI (Dashboard), use the menu to select Appearance -> Header -> Select image -> Browse, and then select Upload. Header images have max size 940px*129px.

= How can I switch the language from English to Russian? =

In the main catalog of your WordPress site find file wp-config.php, open it and change define (‘WPLANG’); to define (‘WPLANG’, ‘ru_RU’);

= How to display Sliding image gallery? =

Sliding default image is not displayed on the site. To display it, you must go to the admin panel-> Appearance -> Slider menu item in the Admin UI, and there choose which category of posts you want to see in the slider, after that set the status draft for posts that you want to display in the slider.
